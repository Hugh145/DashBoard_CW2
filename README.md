Creation of the dashboard   

# Application Description
This application is a dashboard web app built with React and Node.js. It allows users to register, log in, and view interactive data visualizations. The dashboard supports light and dark themes, user management, and displays company and prospect data from a MongoDB database.

Installation Instructions

Clone the Repository 1
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name

Install Backend Dependencies 2
cd Backend_Dashboard
npm install

Install Frontend Dependencies
cd ../HCI_Dashboard
npm install

Set Up Environment Variables of the database and chatgpt api keys 

Create a .env file in both Backend_Dashboard and HCI_Dashboard folders as needed.
Add your MongoDB connection string and any required API keys.

Start the Backend Server
cd ../Backend_Dashboard
npm start

Start the Frontend Development Server
cd ../HCI_Dashboard
npm start

Access the Application
localhost:5173

Open your browser and go to http://localhost:5173 (or the port shown in your terminal).
